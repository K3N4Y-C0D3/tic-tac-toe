**ABRIR O CMD OU O GIT CMD**

> cd "colocar o caminho da pasta"
> dir (mostra o que tem na pasta que você selecionou)
> rd .git /S/Q       (apagar qualquer tipo de git)
> git init 
> git add .   (adicionar todos os arquivos da pasta)
> git commit -m "first commit"   << nome do commit
> git remote add origin "LINK DO REPOSITÓRIO"
> git push -u origin master


obs.: CASO DE ALGUM ERRO USAR ESSES COMANDOS
